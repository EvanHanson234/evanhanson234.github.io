Building an Operating System - Pandos

I have built an operating system in the C programming language called Pandos. This project was a challenging and exciting experience that allowed me to dive into the core of computer systems and learn how they work. I gained a deeper understanding of how operating systems manage and allocate system resources, and how they interact with the underlying hardware.

Reuniting Pets with Owners Application

I am currently working on building an application that will help reunite lost pets with their owners. The application will have both frontend and backend components and will take data from the Facebook API. The aim of this project is to help reunite as many lost pets with their owners as possible, making the process quicker and more efficient.

Designing a Secure Network

In this project, I was tasked with configuring a PaloAlto firewall and designing a secure network for a fictional company. The goal of this project was to protect the company's sensitive data and maintain the confidentiality, integrity, and availability of the network. I gained valuable experience in firewall configuration and network security, and learned about different security protocols and how they can be implemented to protect a network.
